# Images par défaut pour Open Graph

## og-default.jpg

Cette image est utilisée comme image par défaut pour le partage sur les réseaux sociaux (Facebook, Twitter, LinkedIn, etc.) quand aucune image spécifique n'est définie.

**Spécifications recommandées :**
- Taille : **1200x630 pixels** (ratio 1.91:1)
- Format : JPG ou PNG
- Poids : Moins de 8 MB
- Contenu : Logo FunLab + slogan attractif

**Pour créer l'image :**
1. Utilise Canva, Photoshop ou Figma
2. Dimensions : 1200x630px
3. Ajoute le logo FunLab
4. Ajoute le texte : "FunLab Tunisie - Escape Game, Réalité Virtuelle & Laser Game"
5. Utilise les couleurs de la marque (violet/mauve)
6. Exporte en JPG qualité 90%
7. Nomme le fichier `og-default.jpg`
8. Place-le dans `/public/assets/images/`

**Exemple de structure :**
```
+----------------------------------+
|                                  |
|         Logo FunLab              |
|                                  |
|    FunLab Tunisie               |
|    Escape Game, Réalité         |
|    Virtuelle & Laser Game       |
|                                  |
|    Réservez maintenant !        |
|                                  |
+----------------------------------+
```

**Note :** L'image `game-default.jpg` est utilisée pour les jeux qui n'ont pas d'image spécifique.
